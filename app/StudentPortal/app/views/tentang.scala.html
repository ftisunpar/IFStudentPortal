Ketentuan Penggunaan

Layanan ini diberikan secara gratis, tanpa garansi apapun. Pengembang dan pengelola situs tidak bertanggung jawab atas segala kerugian yang mungkin ditimbulkan. Khusus kepada mahasiswa, silahkan memeriksa kembali peraturan resmi dari jurusan / universitas untuk memastikan.

Kebijakan Privasi

Pengembang dan pengelola melakukan upaya terbaik untuk menjaga privasi Anda.

Layanan ini meminta password Anda untuk melakukan login ke situs Portal Akademik Mahasiswa. Program tidak menyimpan password tersebut setelah login dilakukan.

Data mahasiswa diunduh dari situs Portal Akademik Mahasiswa, dan mungkin disimpan oleh server di memori dalam jangka waktu tertentu. Server tidak memberikan akses kepada pihak manapun (termasuk pengembang) atas data tersebut.
